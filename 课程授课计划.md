# 嵌入式 Linux 相机项目授课计划

本课程通过从零实现一个“相机 APP (video2lcd)”的过程，带学生从底层 V4L2 驱动原理一路走到高层 Manager 架构设计。

---

## 📅 课程大纲

### 阶段一：V4L2 编程基础与原始数据采集 (3 课时)
*   **教学目标**：理解 Linux 视频子系统框架，掌握最基本的摄像头数据抓取。
*   **核心内容**：
    1.  **V4L2 架构总览**：介绍 `/dev/videoX` 设备节点，理解 `fops` 和 `ioctl` 的核心地位。
    2.  **核心控制流程**：查询设备能力 (`VIDIOC_QUERYCAP`)、枚举与设置格式 (`VIDIOC_ENUM_FMT`, `VIDIOC_S_FMT`)。
    3.  **缓冲区管理机制**：申请内存 (`VIDIOC_REQBUFS`)、内存映射 (`mmap`)、入队与出队 (`VIDIOC_QBUF`/`DQBUF`)。
    4.  **启动流与捕获**：执行 `VIDIOC_STREAMON` 并读取首帧数据。
*   **实验任务**：编写 `video_test.c`，在命令行捕获一帧 YUV 图像并保存。

### 阶段二：UVC 协议深度剖析与参数控制 (2 课时)
*   **教学目标**：深入理解 USB 摄像头内部工作原理，学会控制硬件参数。
*   **核心内容**：
    1.  **UVC 规范解读**：VideoControl 接口（控制） vs VideoStreaming 接口（数据）。
    2.  **拓扑结构**：理解输入终端 (IT)、处理单元 (PU) 和输出终端 (OT) 的概念。
    3.  **动态控制属性**：利用 `VIDIOC_QUERYCTRL` 查询项，并通过 `VIDIOC_G_CTRL`/`S_CTRL` 调节亮度、对比度等。
*   **实验任务**：在基础实验上增加命令行调节亮度的功能。

### 阶段三：Framebuffer 显示与图像显示原理 (2 课时)
*   **教学目标**：掌握底层显示技术，将像素推送到 LCD。
*   **核心内容**：
    1.  **Framebuffer 驱动原理**：理解 `/dev/fb0` 及其显存映射（Memory Mapping）。
    2.  **LCD 像素格式**：对比 RGB565 与 ARGB8888 的存储差异。
    3.  **画点与刷屏**：如何将摄像头采集的像素坐标映射到 LCD 显存中。
*   **实验任务**：编写 `fb_test.c`，在屏幕指定位置绘制彩色色块。

### 阶段四：项目软件架构设计：Manager 模式 (2 课时)
*   **教学目标**：学习大型嵌入式项目的“高内聚、低耦合”设计思想。
*   **核心内容**：
    1.  **面向对象的 C 编程**：利用结构体封装和函数指针实现“接口”抽象。
    2.  **四大管理模块设计**：
        *   `VideoManager`：支持多种输入源（V4L2、文件等）。
        *   `DispManager`：支持多种显示输出（FB、网络流等）。
        *   `ConvertManager`：处理复杂的格式转换逻辑。
        *   `Render`：渲染层逻辑封装。
*   **实验任务**：分析源码中的 `1th/video2lcd` 架构，画出系统的模块调用框图。

### 阶段五：颜色空间转换与第三方库集成 (3 课时)
*   **教学目标**：解决“看得到但看不清”的问题，实现 YUV/MJPEG 到 RGB 的高效转换。
*   **核心内容**：
    1.  **颜色模型转化**：推导 YUV422 转 RGB 的数学公式及查表法优化。
    2.  **MJPEG 处理**：理解为何需要 MJPEG（高带宽下的图像压缩方案）。
    3.  **libjpeg 移植**：学习如何在交叉编译环境下集成第三方 `jpeglib` 进行硬件压缩流解码。
*   **实验任务**：实现 `3th/video2lcd` 版本，实现在 LCD 上流畅看到彩色视频。

### 阶段六：系统集成、优化与 GUI 拓展 (2 课时)
*   **教学目标**：打磨最终产品，并向进阶 UI 方向引申。
*   **核心内容**：
    1.  **多线程与异步处理**：引入双缓冲或异步 DQBUF 机制减少显示卡顿。
    2.  **工程化构建**：学习 `Makefile.build` 这种大型 Makefile 的递归组织方式。
    3.  **LVGL 拓展（选讲）**：展示如何为相机添加按钮、进度条等图形界面。
*   **实验任务**：部署并运行 `6th_ok` 代码，在开发板上完成完整的相机项目演示。
